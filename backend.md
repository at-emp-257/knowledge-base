1. **Programming languages** – Core syntax, paradigms, and ecosystem of at least one language (e.g., Java, Python, Go, C#).

3. **APIs & web protocols** – REST, GraphQL, HTTP/HTTPS, WebSockets, gRPC.
4. **Databases** – SQL (MySQL, PostgreSQL) and NoSQL (MongoDB, Redis), indexing, normalization, ACID, CAP theorem.
5. **Caching** – In-memory caches (Redis, Memcached), CDNs, cache strategies.
6. **Authentication & authorization** – OAuth, JWT, sessions, SSO, role-based access control.
7. **Security basics** – Encryption, hashing, SQL injection, XSS, CSRF, rate limiting.
8. **Message queues & async processing** – RabbitMQ, Kafka, pub/sub patterns, job queues.
9. **Containerization & orchestration** – Docker, Kubernetes, basic cluster management.
10. **Cloud services** – Compute, storage, networking on AWS, GCP, or Azure.
11. **Version control** – Git workflows, branching, merging.
12. **Testing** – Unit, integration, E2E testing; mocking, TDD basics.
13. **CI/CD** – Pipelines, automated testing, deployment strategies (blue-green, canary).
14. **Monitoring & logging** – Metrics (Prometheus), logs (ELK stack), tracing, alerts.
15. **System design principles** – Scalability, load balancing, microservices, CAP theorem, REST vs gRPC.
16. **Performance optimization** – Database query tuning, concurrency, connection pooling, latency reduction.
17. **Linux/CLI basics** – Shell commands, process management, file permissions.
18. **Networking fundamentals** – TCP/IP, DNS, firewalls, load balancers, proxy servers.
